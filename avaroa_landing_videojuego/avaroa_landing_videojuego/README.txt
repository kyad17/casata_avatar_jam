AVAROA: EL CORAZÓN DEL LITORAL - LANDING PAGE EDITADA

Archivos principales:
- index.html
- css/styles.css
- js/script.js
- assets/img/mapa_litoral.png
- assets/img/eduardo_avaroa.png
- assets/img/escenario_calama.png
- assets/img/fusil_lanza.png
- assets/img/boceto_personaje.jpeg

Cambios realizados:
- Se reemplazó la ilustración básica de la historia por el mapa del Litoral.
- Se reemplazó la imagen de Eduardo Abaroa por el personaje steampunk con fusil y capa.
- Se reemplazó la galería predeterminada por: boceto rojo, escenario de Calama y fusil-lanza.
- Las imágenes se configuraron para verse completas, usando encuadres con object-fit: contain.
- Se mantuvo el botón "Agregar fotografías" y su función de previsualización local.
- Se añadieron animaciones: engranajes, humo/vapor, soldados en movimiento, brillo del cursor, parallax, tarjetas con efecto 3D, barra de progreso y visor ampliado de galería.

Cómo ejecutar:
1. Descomprime el ZIP.
2. Abre la carpeta avaroa_landing_videojuego.
3. Haz doble clic en index.html.
4. También puedes abrirlo desde Visual Studio Code con Live Server.

Nota:
La previsualización de fotos no sube archivos a internet ni los guarda permanentemente. Para guardar fotos de forma real se necesita backend o base de datos.
